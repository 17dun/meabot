# 项目说明文档

## 项目概述

本项目是一个名为“meabot—自媒体自动运营神器”的浏览器扩展程序。其主要功能是分析和评估社交媒体平台“小红书”和“抖音”上的个人账号。

该扩展完全使用原生 JavaScript、HTML 和 CSS 编写，没有依赖任何构建工具或前端框架。

### 核心功能

*   **页面内分析:** 将内容脚本（Content Script）注入到支持平台的用户主页中。
*   **数据抓取:** 提取用户信息（如用户名、ID、粉丝数、获赞数、简介）以及近期发布作品的数据（如点赞数、标题、链接）。
*   **质量评分:** 基于互动率、内容稳定性和受欢迎程度等指标，计算出账号的综合质量得分。
*   **趋势可视化:** 通过折线图展示近期作品的点赞数，帮助用户观察内容表现趋势。
*   **高互动检测:** 与用户的平均水平相比，自动识别并高亮显示点赞数异常高的作品。
*   **UI 注入:** 在被分析的页面上渲染一个浮动侧边栏，用于展示质量得分、趋势图表和高表现作品列表等结果。
*   **用户配置:** 提供一个弹出窗口界面，允许用户切换自动分析模式、调整高赞作品的筛选灵敏度以及设置 API 令牌。

## 构建与运行

这是一个标准的“已解压的”浏览器扩展程序，不需要任何构建步骤。

### 如何运行本扩展：

1.  打开您的 Chromium 内核浏览器（如 Google Chrome, Edge, Arc 等）。
2.  访问扩展管理页面（地址栏输入 `chrome://extensions`）。
3.  打开右上角的 **“开发者模式”** 开关。
4.  点击 **“加载已解压的扩展程序”** 按钮。
5.  在文件选择对话框中，选择本项目的根目录

完成后，扩展程序的图标将出现在浏览器工具栏中。您可以访问小红书或抖音的用户主页，然后通过点击扩展图标或页面上的UI来启动分析。

## 开发规范

*   **`manifest.json`**: 扩展的入口和配置文件。它定义了所需的权限、内容脚本、背景服务工作线程（Service Worker）等。
*   **`content.js`**: 在目标网页上运行的核心逻辑。它负责所有的 DOM 抓取、数据处理以及在页面上注入分析结果面板。该脚本被设计为同时兼容小红书和抖音，并为两个平台编写了不同的处理函数。
*   **`popup.html` / `popup.js`**: 定义了扩展弹出窗口的界面和逻辑。用户可在此处配置 API 令牌、自动分析开关和高赞阈值等设置。
*   **`background.js`**: 在后台运行的服务工作线程。它监听浏览器标签页的更新以及 `chrome.storage` 中 `autoAnalyze` 设置的变更，以便在用户开启自动分析时，在相关页面上自动触发分析流程。
*   **`chrome.storage.sync`**: 用于存储用户的设置（如 API 令牌、自动分析偏好、阈值等），确保这些配置在浏览器会话间保持同步。
